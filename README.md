# Mallika Verma — Portfolio

**Writer who codes.** Personal portfolio built with Next.js, featuring a multi-page editorial design inspired by cinematic Wix templates.

**Live:** [portfolio-web-booster12.vercel.app](https://portfolio-web-booster12.vercel.app)

## Pages

| Route | Description |
|-------|-------------|
| `/` | Home — hero with photo, intro, navigation to all sections |
| `/projects` | Selected work — DriveSafer AI, MindGuard, CureCue, MallikaAI |
| `/experience` | Amazon SDE Intern (May–July 2026) — AI agents, pipelines, Spark migration |
| `/writing` | Medium articles on engineering, internship reflections, and growth |
| `/beyond` | Beyond Code — mentoring, tutoring, Kho Kho, speech competitions |
| `/contact` | Contact form + social links |

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion
- **Smooth Scroll:** Lenis
- **Fonts:** Inter, Playfair Display, JetBrains Mono
- **Icons:** react-icons
- **Deployment:** Vercel

## Design

- Dark theme (#0a0a0a) with gradient accents (sky/purple/pink)
- Multi-page architecture — each section is a separate route
- Editorial layout with alternating image/text sections
- Full-visibility photos (object-contain for collages, object-cover for portraits)
- Cinematic hero with full-body photo
- Interactive hover states, animated navigation buttons
- Custom cursor, Easter egg (Konami code), hidden terminal (Ctrl+`)

## Run Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
